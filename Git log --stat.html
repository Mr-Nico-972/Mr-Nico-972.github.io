Git-log --stat

En tapant git log --stat on a accès au dernières modification commit d'une liste de fichiers , cela indiquent aussi le nombre de fichiers qui ont été modifiés, et le nombre de lignes dans ces fichiers qui ont été ajoutés et supprimés. Il met également un résumé de l'information à la fin.

Syntaxe

$ git log --stat
